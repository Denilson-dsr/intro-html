Sobre as imagens

Trabalhar com imagens no HTML envolve muito mais do que apenas colocar uma foto na pagina

Existe varias formas, tecnicas e recurso para exibir, organizar, otimizar e adptar imagens para diferentes propositos

Formatos mais comuns da imagens

  • JPEG/JPG: fotos e imagens compactadas.
  • PNG: Suporta transparencia
  • WEBp: Formato moderno e leve
  • Gif: Animações simples
  • Avif: Excelente qualidade com tamanho pequeno

Trabalhando com imagens responsivas

O picture é um elemento do HTML usado para imagens responsivas e adapatativas.

Essa tag, permite o seguinte:

  • Trocar a imagem conforme o tamanho da tela
  • Melhorar a perfomance
  • Carregar imagens diferentes para mobile, tablet e desktop
