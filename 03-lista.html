Entendendo Listas do HTML

As listas no html serve para organizar informações relacionadas de forma estruturada

as listas são compostas por itens chamados LI

Orden list - OL

o elemento OL serve para criar lista ordenada, ou seja, com ordem de importancia.

não possui limite de tens

normalmente aparece numerados

lista de compras

  1. Banana
  2. maça
  3. uva

Ordered List - OL

O elemento OL serce para criarmos uma lista ordenada, ou seja, com ordem, importância.

Normalmente aparece numerada

Lista de compras

  2. Maça
  3. Mamão
  4. uva

Unordered List

Serve para criar listas sem ordem especifica. Normalmente aparece com bolinhas (bullets)

Exemplo rápido

  • Amor
  • Estudar
  • Lavar louça
