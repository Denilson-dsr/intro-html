Definição de link:

Um link é um elemento que permite navegar de uma pagina ou recurso

No html, a gente usa links para:

  • Abrir um site externo
  • levar usuario para outra pagina
  • iniciar download de arquivos

Conhecendo elemento

o elemento utilizado para criar link é a tag (anchor)

Acesse o site do google

target: define onde o conteudo será aberto quando o user clicar nele

_blank : abre uma nova aba

_parent: usamos quando a pagina está em um iframe, o link abre um frame no pai.
